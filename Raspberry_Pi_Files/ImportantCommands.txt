
Some Important Commands For The Terminal:
*The RPI uses the Bash shell/scripting language for the terminal
**IP for the Pi is 10.0.0.192

Get IP Address:
hostname -I

Check Temp:
/opt/vc/bin/vcgencmd measure_temp

Show free space available:
df -h

Shutdown:
sudo shutdown -h now

Acess Settings:
sudo raspi-config

Exit Terminal:
exit

List All LAN Networks:
sudo iwlist wlan0 scan

Update Pi Software:
sudo apt-get update 
sudo apt-get upgrade
sudo apt-get dist-upgrade 

Install Packages:
sudo apt install

Set Up Shell From bash_profile:
source ~/.bash_profile

Show Directory Tree:
TREE

Get File From Web:
wget 

Kill Suspended Processes:
kill %process_number

Remove a non-empty directory:
rm -r

Show Pin Layout:
pinout
